\subsection*{Intended Learning Outcomes - Lecture 03}

The student should be able to:
\begin{itemize}
\item understand the differences between endogenous, exogenous input and exogenous random variables;
\item understand how hard interventions can be modeled using structural equations;
\item understand how soft interventions can be modeled using structural equations;
\item understand how intervention variables can be used to model different interventions in the same causal model;
\item propose an appropriate structural causal model for a given real-world system, and to appropriately model interventions on the system using the model;
\item know the formal definitions of structural causal model, solutions, potential outcomes, solution functions, and the different variants of hard interventions;
\item understand the implicit assumptions made when using a structural causal model.
\end{itemize}
