\textbf{Statement.} Almost-sure do-calculus (simplified): four rules
(insertion/deletion of observation, action/observation exchange,
insertion/deletion of action, deletion of input).

\textbf{Analysis.}
Rules 1--3 follow directly from Thm.\ \texttt{thm:as-do-calculus-1}
with the appropriate choices of intervention decompositions.  $\checkmark$

Rule~4 (deletion of input): defines
$P(X_A|X_C,\doit(X_D,\cancel{X_{J\setminus D}}))
 := Q(X_A|X_C,X_D)$
from Prp.\ \texttt{prp:do-calculus} point~3 with $B=\emptyset$.
The graphical condition $A \perp_G J\mid C\cup D$ is used to ensure
the kernel is independent of $X_{J\setminus D}$.
The a.s.\ equality follows from essential uniqueness + absolute continuity.
$\checkmark$

\textbf{Verdict: CORRECT.}

\section{Proof 3: Prp.\ \texttt{prp:do-calculus} (lines 79--217)}
%% ========================================================================

\textbf{Statement.} Do-calculus existence and uniqueness: under
$i$-$d$-separation conditions, there exist ``canonical'' conditional Markov
kernels $Q$ that are simultaneously versions of various conditional kernels
across all intervention decompositions.  Three parts (rules 1--3).

\textbf{Analysis.}
\begin{enumerate}[leftmargin=*]
  \item \textbf{Part 1 (lines 82--109):} Applies GMP to $M_{\doit(D)}$,
    obtains factorization, marginalizes over $X_{B_1}$ for each $B_2\subseteq B$.
    Uniqueness via essential uniqueness of disintegration.  $\checkmark$
  \item \textbf{Part 2 (lines 112--166):} GMP gives $X_A\perp X_{I_B}|X_B,X_C,X_D$.
    Specializing $X_{I_{B_1}}=\star$, $X_{I_{B_2}}=x_{B_2}$ and marginalizing
    the deterministic $X_{B_2}$ yields the factorization for each decomposition.
    Uniqueness via Lem.\ \texttt{lem:int-null-set}.  $\checkmark$
  \item \textbf{Part 3 (lines 169--217):} Similar to Part~2 but conditioning
    on $C\cup D$ instead of $B\cup C\cup D$.  $Q$ depends on $(x_C,x_D)$
    only.  Uniqueness argument traces through all $x_{I_B}$ values.  $\checkmark$
\end{enumerate}

\textbf{Verdict: CORRECT.}

\section{Proof 4: Thm.\ \texttt{thm:as-do-calculus-1} (lines 221--277)}
%% ========================================================================

\textbf{Statement.} Almost-sure do-calculus (detailed): for finitely many
versions simultaneously, there exists a common null set $N$ outside which all
agree with~$Q$ from Prp.\ \texttt{prp:do-calculus}.

\textbf{Analysis.}
For each rule, the proof takes the finite collection of versions, shows each
deviates from~$Q$ on a null set (by essential uniqueness), lifts these to
$P(X_B,X_C|\doit(\cdots))$-null sets (via cylinder/product structure), and
converts to $\mu$-null sets via absolute continuity.
The finite union is still null.  $\checkmark$

\textbf{Verdict: CORRECT.}

$P(X_C|\doit(X_{J\cup V\setminus C})) = P(X_C|\doit(X_{\Pa^G(C)\setminus C}))$.

no ancestors).  $\checkmark$

\section{Proof 11: Lem.\ \texttt{lem:id-d-sep-3} (lines 290--337)}
%% ========================================================================

\textbf{Statement.} Graph-theoretic lemma for the interventional ordered local
Markov property: $\{v\}\perp V_< | \Pa^G(D_\le)\cup D_<$ in
$G_{\doit(I_{V\setminus D_\le})}$.

\textbf{Analysis.}
Proof by contradiction.  Case analysis on the first node $\tilde w$ outside
$D_\le$ on a supposed unblocked walk:
\begin{itemize}
  \item $\tilde w\in\Pa^G(D_\le)\setminus D_\le$: non-collider in
    conditioning set, blocked.  $\checkmark$
  \item $\tilde w\in V_>$: no colliders in $V_>$ can be ancestors of $F$,
    blocked.  $\checkmark$
  \item $\tilde w\in V_<\setminus D_\le$ via directed edge $v_k\to\tilde w$:
    contradicts topological order.  $\checkmark$
  \item $\tilde w\in V_<\setminus D_\le$ via bidirected edge: contradicts
    $\Dist^{G_\le}(D_\le)\subseteq D$.  $\checkmark$
\end{itemize}

\section{Proof 13: Lem.\ \texttt{lem:int-anc} (lines 410--424)}
%% ========================================================================

\textbf{Statement.} For ancestral $A=\Anc^{[V]}(A)$:
$Q^{[A]} = P(X_A|\doit(X_{J\cup V\setminus A})) = P(X_A|\doit(X_J))$.

\textbf{Analysis.}
Shows $i$-$d$-separation $A\perp I_{V\setminus A}|J$ in
$G_{\doit(I_{V\setminus A})}$.  Key: $\Anc^G(A)\setminus A\subseteq J$
(by ancestrality), so any parent outside~$A$ is in~$J$ and blocks the walk.
Colliders cannot be ancestors of~$J$ (input nodes have no ancestors).
$\checkmark$

\textbf{Verdict: CORRECT.}
