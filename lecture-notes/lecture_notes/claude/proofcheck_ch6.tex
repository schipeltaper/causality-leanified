\section{Proof 1: Lem.\ \texttt{lem:M-as\_to\_rv-as} (lines 603--612)}
\textbf{Statement.} $M$-a.s.\ properties lift to a.s.\ properties of random
variables whose $W$-marginal is $P_M$.

\textbf{Analysis.} Fixing $x_J$, the $M$-null set gives a $P_M$-null set in
$\Xcal_W$.  Since $X_W\sim P_M$, the property holds a.s.  $\checkmark$

\textbf{Verdict: CORRECT.}

\section{Proof 2: Lem.\ \texttt{lem:M-as\_to\_crv-as} (lines 621--630)}
\textbf{Statement.} Same as Proof~1 but for conditional random variables with
constant $W$-marginal kernel $P_M$.

\textbf{Analysis.} The kernel $K(X_W|X_J)=P_M$ assigns zero measure to the
failure set for every $x_J$.  $\checkmark$

\textbf{Verdict: CORRECT.}

\section{Proof 3: Prp.\ \texttt{prp:using\_solution\_functions} (lines 677--703)}
\textbf{Statement.} Solution functions yield potential outcomes, solutions, and

\textbf{Analysis.} Applies Lemmas 1--2 to lift $M$-a.s.\ structural equations
to a.s.\ statements about $(g(x_J,X_W),X_W)$.  $\checkmark$

\textbf{Verdict: CORRECT.}

\section{Proof 4: Prp.\ \texttt{prp:scm\_partial\_solvability} (lines 777--817)}
\textbf{Statement.} Characterization of essential unique solvability w.r.t.\ $L$
via measurable partial solution functions.

\textbf{Analysis.}
$(\Rightarrow)$: Constructs $g^{[L]}$ on $\tilde N^c$ using uniqueness of the
fixed point, extends measurably via \cite{Kec95} 14.12.  $\checkmark$

$(\Leftarrow)$: Claims the complement of $U^{[L]}$ is contained in a
``measurable $M$-null set,'' but the definition of $M$-a.s.\ only guarantees
an $M$-null set (not necessarily measurable).

\textbf{Typo:} Line~798 has $x_V^0$ where $x_L^0$ is intended.

\textbf{Verdict: GAP} (minor foundational subtlety in the backward direction;
resolves if $M$-a.s.\ is defined via measurable null sets).
