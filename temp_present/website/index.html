Definition 3.1

Conditional Directed Mixed Graph (CDMG)

Formalised No proof (definition)
Statement (lecture notes)

A conditional directed mixed graph (CDMG) \(G\)—per definition—consists of two (disjoint) sets of vertices (also called nodes):

  1. \(J\), whose elements are called input nodes,
  2. \(V\), whose elements are called output nodes,

and two (disjoint) sets of edges:

  1. \(E \subseteq (J \cup V) \times V\), the set of directed edges,
  2. \(L \subseteq V \times V / ((v_1, v_2) \sim (v_2, v_1))\), the set of bidirected edges,
  3. with: \((v_1, v_2) \in L \implies v_1 \neq v_2 \land (v_2, v_1) \in L\).
Formalisation CDMG.lean · Causality.CDMG 
structure CDMG (α : Type*) where
  /-- The set of *input* nodes. -/
  J : Set α
  /-- The set of *output* nodes. -/
  V : Set α
  /-- Inputs and outputs are disjoint sets of vertices. -/
  disjoint_JV : Disjoint J V
  /-- The set of *directed* edges. -/
  E : Set (α × α)
  /-- Directed edges originate at any node and terminate at an output. -/
  E_subset : E ⊆ (J ∪ V) ×ˢ V
  /-- The set of *bidirected* edges, encoded as a symmetric, irreflexive
  subset of `V × V` — the unfolded version of the LN's quotient
  `V × V / ∼`. -/
  L : Set (α × α)
  L_subset : L ⊆ V ×ˢ V
  L_irrefl : ∀ ⦃v₁ v₂ : α⦄, (v₁, v₂) ∈ L → v₁ ≠ v₂
  L_symm   : ∀ ⦃v₁ v₂ : α⦄, (v₁, v₂) ∈ L → (v₂, v₁) ∈ L
  /-- Directed and bidirected edges are disjoint. -/
  disjoint_EL : Disjoint E L

Design choices & equivalence with the LN
  • Polymorphic vertex type α : Type*. The LN does not commit to finiteness or countability at this layer; keeping α fully polymorphic lets every later chapter pick its ambient vertex type without forcing changes here.
  • Set α for J and V. The LN literally says "two (disjoint) sets of vertices", and every downstream definition reads j ∈ J / v ∈ V rather than indexing into a list.
  • Bidirected edges as a symmetric irreflexive subset of V × V rather than the literal quotient — propositionally equivalent, but the quotient carrier forces explicit Quotient.mk/Quotient.lift at every use site.
  • structure, not class or abbrev. A specific CDMG is data, not a canonical instance attached to α. structure also gives projection notation G.J, G.V, … matching the LN's "Let G = (J,V,E,L)" pattern.
  • No SimpleGraph base. SimpleGraph is single-sorted, symmetric, irreflexive on a single vertex type — none of which match a CDMG.
